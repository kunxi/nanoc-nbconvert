# nanoc-nbconvert

This provides a [IPython notebook](http://ipython.org/notebook) filter for [nanoc](http://nanoc.ws) using [nbconvert](http://ipython.org/ipython-doc/rel-1.0.0/interactive/nbconvert.html).

## Installation

`gem install nanoc-nbconvert`

## Usage

```ruby
filter :nbconvert
```
